Deploy the Currency Converter

1.Extract project2.rar to your tomcat-path/webapps/app-name/ folder.
2.Restart tomcat
3.Enjoy!!


***If you need to change any .java file, you can do it. The .java files are located at /WEB-INF/src inside project2.rar. Compile using javac -classpath tomcat-path/lib/servlet-api.jar, or any other IDE you wish, like netbeans or eclipse. Put the .class files at tomcat-path/webapps/app-name/WEB-INF/classes.***
